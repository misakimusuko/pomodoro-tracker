# 🍅 ポモドーロ & タスクトラッカー

シンプルなポモドーロタイマーとタスク管理を組み合わせたWebアプリです。HTML / CSS / JavaScriptのみで動作し、ビルドツールは不要です。

## 特徴

- 作業(25分)/ 小休憩(5分)/ 長休憩(15分)の3モードを切り替え可能なタイマー
- 4回の作業セッションごとに自動で長休憩を提案
- タスクの追加・完了チェック・削除
- タスクを選択して「アクティブなタスク」に設定すると、完了したポモドーロ数がそのタスクに記録される
- 完了したポモドーロ数・タスクの進捗はブラウザの`localStorage`に保存され、再読み込み後も保持される

## 使い方

ビルドや依存関係のインストールは不要です。`index.html`をブラウザで開くだけで動作します。

```bash
# 例: ローカルで簡易サーバーを立てて開く場合
npx serve .
```

## ファイル構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | アプリのマークアップ |
| `style.css` | スタイル定義 |
| `script.js` | タイマー・タスク管理のロジック |
